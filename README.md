---

##  Installation

### Prerequisites

- Python 3.10
- Pygame library

### Steps

**Clone the Repository**:

```bash
git clone https://github.com/Amaanish/Snake-Game.git
cd Snake-Game
```
Then

```bash
pip install pygame
```
Run the game in VSCode
